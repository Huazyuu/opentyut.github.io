# 学习指南概览

本指南面向太原理工大学 **计算机相关专业大一新生**，帮助你从零开始规划学习路线。

## 这个指南是什么

- **不是**官方教学大纲的替代品
- **是**学长们走过弯路后的经验总结
- **是**课堂之外的自学路线图

## 如何使用

每个学期的页面包含：

| 内容 | 说明 |
|------|------|
| **课程列表** | 该学期的主要课程及难度评估 |
| **学习建议** | 学长对该学期课程的学习心得 |
| **推荐资源** | 网课、教材、博客、项目等 |

## 关于计算机学习

> 大学计算机教育教的是"计算机科学"，不是"写代码"。你会在课堂上学到操作系统、数据结构、编译原理，但不会有人手把手教你写一个完整的 Web 应用。这个差距需要你自己填补。

**大一最重要的是两件事：**

1. **培养自学能力** —— 这是整个职业生涯最重要的技能
2. **打好编程基础** —— 至少熟练掌握一门语言

::: tip 提示
不要让绩点成为你唯一的追求。多写代码，多做项目，多看文档。
:::
